ABBEY
- A New Female Player Model for Unreal Tournament

AUTHOR. : Roger [666] Bacon

Abbey Chase is the lead character from J.Scott Campbells action comic 'Dangergirl'

USAGE: Run the .UMOD and enter the location of your Unreal Tournament directory. The player model will then install. Select the new mode in the player setup screen. Choose:
Class : Female -666Abbey

I originally intended for this to be just a general female player model that would be easily customisable for other skinners. Most player models are fairly oddly shaped and dont lend themselves well to custom skins, but Abbey is a fairly plain shape and should be good for a variety of character skins. (Although she is fairly stylised in comic book proportions).

She became Abbey as I was reading Dangergirl at the time and realised she had the same proportions. :)

This is the first model I have created with extra animation elements (two for the hair and two for her ample 'attributes')
Extra cheers for this model go out especially to Alphawolf and the chaps on the Polycount UBB for helping me add extra bits to Biped. (big up to KingOfDaveness, John Sheffield, and Alcor)


*IMPORTANT!*
USING ABBEY ON A NETWORK (LAN) OR INTERNET
If you run a server using this model, please add these lines to unrealtournament.ini under all the others in [engine.gamengine] or else you will get invisable characters and green skins.

ServerPackages=6Abbey
ServerPackages=AbbeySkins

If you are concerned about missing skin error messages in the Unrealtournament.log file, dont worry as they are harmless. The only way I could get a single skin model to work was to make it think it had two.
